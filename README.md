# 🧠 Guía de Ejercicios Básicos con JavaScript

Repositorio creado para la práctica de los **fundamentos de la lógica de programación en JavaScript**, siguiendo la guía de ejercicios proporcionada en clase.

---

## 🎯 Propósito

El objetivo de esta práctica es fortalecer habilidades básicas de programación, trabajando con:

* Variables (`let`, `const`)
* Operaciones aritméticas
* Estructuras condicionales (`if`, `else`, `switch`)
* Bucles (`for`, `while`)
* Uso de `console.log()`

Todo el desarrollo se realiza **sin manipulación del DOM**, enfocado únicamente en la lógica.

---

## 📘 Descripción

En este repositorio se resuelven 10 ejercicios progresivos que permiten comprender los conceptos fundamentales de JavaScript.

Cada ejercicio aborda una temática específica, desde la declaración de variables hasta el uso de ciclos y estructuras de control.

## ▶️ Cómo ejecutar

Puedes ejecutar el archivo de dos formas:

### 🔹 Opción 1: Con Node.js

```bash
node ejercicios.js
```

### 🔹 Opción 2: En el navegador

1. Abre el navegador
2. Presiona `F12` para abrir la consola
3. Copia y pega el código
4. Ejecuta y observa los resultados
